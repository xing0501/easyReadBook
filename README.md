# 悅讀聽書
一款簡單、容易操作的電子書閱讀app，具有語音唸書功能。
支援 EPUB 格式，兼具閱讀及語音朗讀的閱讀器
## 功能特色
隨時隨地，開啟您的私人朗讀書房。<br/>
「悅讀聽書」是一款專為 iOS 使用者設計的極簡電子書閱讀器。無論是沉浸在文字的海洋，還是在通勤時透過聲音「閱讀」，這款 App 都能提供最舒適的體驗。<br/>
【核心功能】<br/>
多元格式支援： 完美相容 EPUB 與 TXT 文件，輕鬆導入您的個人藏書。<br/>
AI 語音朗讀： 內建高品質語音引擎，支援多種語速與音調調整，讓閱讀變成聽覺饗宴。<br/>
深度自定義： 自由調整字體、字號、背景顏色（包含護眼米色與夜間模式）及邊距，打造專屬閱讀介面。<br/>
直覺操作： 點擊翻頁、長按標記，以及專為閱讀設計的雙擊點讀功能。<br/>
進度同步： 自動記錄閱讀位置與書籤，換個時間也能無縫接軌。<br/>
簡約而不簡單，讓閱讀回歸純粹。<br/>
